```markdown
# Metalytic-style Wiki — Starter

This is a Jekyll site using the Minimal Mistakes theme configured for GitHub Pages.

How to edit:
- Replace text pages in index.md, description.md, engineering.md, results.md
- Replace placeholders in assets/img/ with your own images (keep filenames or update page links)

Publish:
- Ensure repository is Public.
- Settings → Pages → Branch: main, Folder: / (root)

Local preview (optional):
- Install Ruby and Bundler
- Run: `bundle install` then `bundle exec jekyll serve`
```
